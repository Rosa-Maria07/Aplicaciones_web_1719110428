TAGS de HTML5


< !DOCTYPE html >

Es una declaracion que define que el documento es HTML5, nos sirve para indicar que nuestro documento está escrito siguiendo la estructura determinada.

< html >

Es el elemento raiz de una pagina web, se define mediante una etiqueta de inicio, hasta la etiqueta final en la cual todos los documentos html constan de elementos html anidados. HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet

< head >

Es la primera de las dos partes en que se estructura un documento HTML. Reside información acerca del documento, y generalmente no se ve cuando se navega por él,

< title >< /title >

Es el factor más importante de la optimización en la pagina básica de una Web. Este elemento especifica un titulo para la pagina html

< body >

Este elemento define el cuerpo del documento y es un contenedor para todos los contenidos visibles como encabezados, parrafos, imagenes, hipervinculos, tablas, listas, etc. Contiene una etiqueta de inicio y una etiqueta de finalizacion. El cuerpo (body) del documento html es normalmente lo más importante.

< h1 >< /h1 >

Es el encargado de definir un encabezado, el cual consta de una etiqueta de inicio y una etiqueta definalizacion, asi mismo define el encabezado mas importante o principal.

< p >< /p >

Es el elemento de un parrafo, un parrafo siempre comienza en una nueva linea y los navegadores agregan automaticamente un espacio en blanco antes y despues de un parrafo.
